 # 바닥부터 만드는 LLM
 
 #### LLM 만들때는
 1. 사전 훈련(pretraining)으로 일반적인 언어 능력을 가르친 후에 
 2. 미세 조정(fine tuning) 단계에서 특정 업무에 적응

#### 이것이 기본이 됨. 여기에
3. 데이터베이스(+인터넷) 검색 기능을 추가
#### 하면 지식의범위와 정확성을 높일 수 있음.
#### 생각을 거듭하여 더 깊이있는 결론을 이끌어 내듯이
4. LLM도 내부적으로 질의를 반복하여 더 좋은 결론을 도출
#### 하도록 만들 수 있음
